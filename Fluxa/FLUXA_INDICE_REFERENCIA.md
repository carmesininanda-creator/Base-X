## 2. MODELO DE NEGÓCIO (v21 — OFICIAL)

### 2.1 Regra fundamental
- **O produtor NÃO paga taxa sobre vendas**
- Quem paga: **cliente final** (taxa de conveniência transparente)
- Produtor recebe: **100% do GMV**
- Modelo comparável: iFood, Uber, Sympla

### 2.2 Taxa
- **Taxa única: 5,9% para TODOS os planos**
- Diferencial entre planos = nível de inteligência do Syntropy™, NÃO a taxa
- Custo gateway: ~1,74% (mix 70% Pix MP 0,99% + 30% Cartão Stripe 3,49%)
- Margem líquida na taxa: ~4,16%

### 2.3 Planos (Junho 2026)

| Plano | Preço | Taxa | Syntropy™ |
|---|---|---|---|
| Starter | Grátis | 5,9% | Relatórios básicos pós-evento |
| Single Event | R$ 697–1.297/evento | 5,9% | Planejamento Inteligente + Mission Control |
| Pro | R$ 2.497/mês | 5,9% | Operação Preditiva + IA completa |
| Enterprise | Sob consulta (≥ R$ 7.500/mês) | 5,9% | Inteligência Estratégica + API + DaaS |

### 2.4 Motores de receita
1. **Taxa de conveniência (5,9%)** — escala com volume
2. **Mensalidade SaaS** — receita recorrente (R$ 0,4M Ano 1 → R$ 14M Ano 5)
3. **Data-as-a-Service (mês 18+)** — dados agregados/anonimizados, margem ~95%
4. **Patrocínios inteligentes (futuro)** — não incluído nas projeções base

## 16. REGRAS OPERACIONAIS

1. **Nunca inventar dados** — se não está neste índice, buscar no documento fonte antes de usar.
2. **Taxa é 5,9% para todos os planos** — diferencial é Syntropy, não taxa.
3. **Enterprise é "sob consulta (≥ R$ 7.500/mês)"** — nunca um valor fixo inventado.
4. **Projeção oficial: R$ 1,3M → R$ 49,9M (base)** — não usar sequências antigas.
5. **LTV/CAC é ~6x** — não 22,5x nem 24x.
6. **Margem bruta ~71–73%** — não 65% nem 78%.
7. **Cap Table tem 2 versões (A e B)** — sempre especificar qual.
8. **Fluxa NÃO é carteira, NÃO custodia recursos** — gateway autorizado processa.
9. **Documentos internos nunca saem da empresa** — especialmente Engines, Arquitetura Cognitiva, Code Atlas.
10. **Para investidor: narrativa > documentação técnica** — vender mercado + execução + moat.
