## Deploy no Streamlit Cloud

1. Acesse [share.streamlit.io](https://share.streamlit.io)
2. Conecte o repositório `carmesininanda-creator/Base-X`
3. Branch: `main`
4. Main file path: `giulieta/app.py`
5. Em "Advanced settings", adicione o secret:
   ```
   OPENAI_API_KEY = "sua-chave-aqui"
   ```
6. Clique em **Deploy**
